Easily duplicate any post or page, along with all its terms and taxonomies, with just a click!

== Description ==
Duplicate Post or Page with Terms is a simple WordPress plugin that allows users to duplicate any post or page, including all assigned terms and taxonomies (such as categories, tags, or custom taxonomies). This feature is perfect when you need to clone existing content and ensure all associated metadata is carried over to the duplicated post.

### Features:
- One-click duplication of posts and pages
- Copies assigned categories, tags, and custom taxonomies
- Customizable duplication behaviour
- Seamless integration with the WordPress admin interface

== Installation ==
1. Download the plugin ZIP file.
2. In your WordPress admin dashboard, go to **Plugins > Add New**.
3. Click **Upload Plugin**, then **Choose File**, and select the plugin ZIP file.
4. Click **Install Now** and then **Activate** the plugin.

== Frequently Asked Questions ==

= Will this plugin duplicate all the terms and taxonomies? =
Yes, the plugin will duplicate all associated terms (like categories and tags) and taxonomies for the post or page being duplicated.

= Can I customize the duplication behavior? =
Yes, you can customize the duplication behaviour. Navigate to "Settings > Content Duplicator" and there you will get the options.
